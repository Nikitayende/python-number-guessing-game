# Number Guessing Game

A simple Python game where the user tries to guess a randomly generated number.

## Features

- Random number generation
- User input handling
- Too High / Too Low hints
- Unlimited attempts until the correct answer is guessed
- Attempt counter

## Concepts Used

- Python Variables
- Input / Output
- While Loop
- If-Elif-Else Statements
- Comparison Operators
- Random Module

## Technologies Used

- Python 3
- VS Code
